# 2. Overview
<br>

<ul>
  <li>나만의 옷장을 기반으로 OpenAi 연동 및 추천 시스템 구성</li>
  <li>인터넷 옷 쇼핑몰의 옷 데이터를 크롤링 하여 Cloth DB 구축</li>
  <li>크롤링 데이터를 통한 티쳐블 머신 Machine Learning 모델 구축으로 옷 스타일 예측</li>
</ul>
